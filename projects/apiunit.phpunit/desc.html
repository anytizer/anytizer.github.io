The REST API testing is now made very easy, using PHPUnit and my relay class based on curl.

Technical Support is available to donors. Educates on how to install, embed and use it in your projects.

The nicest part of the project is that it can "host" unit tests for your own projects, that are not the part of this project directly. Consider adding a soft link to your API Unit Tests inside this project. For windows, use mklink, for linux use ln.
